# <a name="cursors-event-category"></a>Категория событий Cursors
  Категория событий **Cursors** содержит классы событий, используемых для контроля действий курсоров.  
  
## <a name="in-this-section"></a>в этом разделе  
  
|Раздел|Description|  
|-----------|-----------------|  
|[Класс событий CursorClose](cursorclose-event-class.md)|Описывает события закрытия курсора, возникающие в API-курсорах.|  
|[Класс событий CursorExecute](cursorexecute-event-class.md)|Описывает события выполнения курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorImplicitConversion](cursorimplicitconversion-event-class.md)|Описывает события неявного преобразования курсоров, возникающие в API-курсорах или курсорах [!INCLUDE[tsql](../../includes/tsql-md.md)] .|  
|[Класс событий CursorOpen](cursoropen-event-class.md)|Описывает события открытия курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorPrepare](cursorprepare-event-class.md)|Описывает события подготовки курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorRecompile](cursorrecompile-event-class.md)|Описывает события перекомпиляции курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorUnprepare](cursorunprepare-event-class.md)|Описывает аннулирующие курсор события, возникающие в API-курсорах.|
