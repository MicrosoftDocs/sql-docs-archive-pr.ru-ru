# <a name="advancedproperties-property-sqlservice-class"></a>Свойство AdvancedProperties (класс SqlService)
  Возвращает массив ссылок на объекты, содержащие дополнительные свойства для объекта `SqlService`.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.AdvancedProperties [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 Объект [класса SqlService](sqlservice-class.md) , представляющий службу.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Массив объектов [класса класс sqlserviceadvancedproperty](../sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) , содержащих дополнительные свойства `SqlService` объекта.
