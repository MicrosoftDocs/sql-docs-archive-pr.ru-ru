# <a name="serversettingsgeneralflag-class"></a>Класс ServerSettingsGeneralFlag
  Класс [ServerSettingsGeneralFlag](serversettingsgeneralflag-class.md) представляет настройки общего флага на экземпляре [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Этот класс может использоваться для следующих операций.  
  
-   Просмотр имени флага.  
  
-   Просмотр и изменение значения флага.
