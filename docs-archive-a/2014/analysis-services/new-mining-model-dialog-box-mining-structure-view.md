# <a name="new-mining-model-dialog-box-mining-structure-view"></a>Диалоговое окно «Создание модели интеллектуального анализа данных» (представление структуры интеллектуального анализа данных)
  Используйте диалоговое окно **Создание модели интеллектуального анализа данных** , чтобы добавить новую модель в структуру интеллектуального анализа данных.  
  
## <a name="options"></a>Варианты  
 **Имя модели**  
 Введите уникальное имя для новой модели интеллектуального анализа данных. Имя должно быть уникальным в базе данных, а не только в структуре интеллектуального анализа данных.  
  
 **Имя алгоритма**  
 Выберите алгоритм, на котором должна быть основана новая модель интеллектуального анализа данных.
