# <a name="specify-object-counts-aggregation-design-wizard"></a>Определение счетчиков объектов (мастер статистических схем)
  Для автоматического вычисления количества объектов в кубе или ввода оценочного числа вручную используется страница **Определение счетчиков объектов** . С помощью счетчиков объектов мастер статистических схем оценивает требования к хранилищу.  
  
## <a name="options"></a>Варианты  
 **Объекты куба**  
 Выводит измерения и атрибуты куба. Показаны только те атрибуты, свойства которых не `AggregationUsage` заданы на `None` странице **Анализ использования статистической обработки** мастера, поскольку это единственные атрибуты, требующие указания счетчиков.  
  
 **Оценочное число**  
 Выводит оценочное число строк группы мер и оценочное число элементов атрибута в измерениях базы данных. Значения оценочного числа можно ввести вручную или вычислить. Чтобы вычислить значения счетчиков, введите `0` в поле и нажмите кнопку **количество**. Поля, где уже отображаются оценочные числа, не обновляются.  
  
 **Количество секций**  
 Введите оценочное число строк группы мер и оценочное число элементов атрибута в секциях (необязательно).  
  
 **Count**  
 Вычисляет и повторно заполняет значения в столбце **Оценочное число** для всех пустых полей. Поля, где уже отображаются оценочные числа, не обновляются.
