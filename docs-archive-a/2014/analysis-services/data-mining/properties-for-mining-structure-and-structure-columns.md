# <a name="properties-for-mining-structure-and-structure-columns"></a>Свойства структур интеллектуального анализа данных и их столбцов
  Можно задать или изменить свойства структуры интеллектуального анализа данных, а также связанных с ней столбцов и вложенных таблиц, используя вкладку **Структура интеллектуального анализа данных** конструктора интеллектуального анализа данных. Свойства, устанавливаемые на этой вкладке, распространяются на каждую модель интеллектуального анализа данных, связанную с этой структурой.  
  
> [!NOTE]  
>  Если в структуре интеллектуального анализа данных было изменено любое значение, даже метаданные, например имя или описание, то будет необходимо провести повторную обработку структуры интеллектуального анализа данных и ее моделей перед просмотром или созданием запросов для них.  
  
## <a name="properties-of-mining-structures-and-mining-structure-columns"></a>Свойства структур интеллектуального анализа данных и их столбцов  
 В следующей таблице описаны свойства структуры интеллектуального анализа данных и столбцы структуры интеллектуального анализа, относящиеся к интеллектуальному анализу, которые можно просмотреть или настроить на вкладке **Структура интеллектуального анализа** . Чтобы просмотреть или настроить эти свойства, щелкните правой кнопкой мыши элемент в представлении в виде дерева и выберите пункт **Свойства**.  
  
-   Чтобы просмотреть свойства структуры, щелкните заголовок структуры интеллектуального анализа данных.  
  
-   Чтобы просмотреть свойства столбца или вложенной таблицы, щелкните имя столбца.  
  
### <a name="properties-of-the-mining-structure"></a>Свойства структуры интеллектуального анализа данных  
  
|Свойство|Описание|  
|--------------|-----------------|  
|**CacheMode**|Указывает, следует ли кэшировать варианты, используемые при обучении, или их необходимо удалять после завершения обучения.<br /><br /> Примечание. для этого свойства необходимо задать значение, `KeepTrainingCases` чтобы включить детализацию и контроль.|  
|**Параметры сортировки**|Задает параметры сортировки по умолчанию для столбца. Если параметры сортировки не указаны, используются параметры сортировки сервера.|  
|**Описание**|Описывает структуру интеллектуального анализа данных. Рекомендуется, чтобы в описании была указана цель структуры, а также строение данных, находящихся в ней.|  
|**ErrorConfiguration (по умолчанию)**|Указывает параметры для особой обработки ошибок, если они возникнут.|  
|**HoldoutMaxCases**|Указывает максимальное число вариантов структуры, которые можно зарезервировать в качестве набора проверочных данных.  Если заданы значения и в поле **HoldoutMaxCases** , и в поле **HoldoutPercent**, эти условия объединяются.<br /><br /> Примечание. для задания этого свойства <xref:Microsoft.AnalysisServices.MiningStructure.CacheMode%2A> необходимо задать значение `KeepTrainingCases` .|  
|**HoldoutPercent**|Указывает максимальный процент вариантов структуры, которые можно зарезервировать в качестве набора проверочных данных. Если заданы значения и в поле **HoldoutMaxCases** , и в поле **HoldoutPercent**, эти условия объединяются.<br /><br /> Примечание. для задания этого свойства <xref:Microsoft.AnalysisServices.MiningStructure.CacheMode%2A> необходимо задать значение `KeepTrainingCases` .|  
|**HoldoutSeed**|Определяет начальное значение для инициализации секционирования контрольного проверочного набора, чтобы гарантировать, что проверочный набор данных может быть создан повторно.<br /><br /> Примечание. для задания этого свойства <xref:Microsoft.AnalysisServices.MiningStructure.CacheMode%2A> необходимо задать значение `KeepTrainingCases` .|  
|**Идентификатор**|Уникальный идентификатор структуры интеллектуального анализа данных.<br /><br /> Имя, присвоенное структуре интеллектуального анализа данных при ее создании, используется в качестве идентификатора. Если впоследствии это имя будет изменено методом введения нового имени в свойство `Name`, то новое имя будет использоваться только в качестве псевдонима, а идентификатор не изменится.|  
|**Язык**|Язык для заголовков в структуре интеллектуального анализа данных.|  
|`Name`|Задает имя или псевдоним для структуры интеллектуального анализа данных.<br /><br /> Если изменить значение свойства Name, то новое имя будет использоваться только как заголовок или псевдоним. Идентификатор для структуры интеллектуального анализа данных не изменится.|  
|**Source**|Отображает имя источника данных и его тип.|  
  
### <a name="properties-of-the-mining-structure-columns"></a>Свойства столбцов интеллектуального анализа данных  
  
|Свойство|Описание|  
|--------------|-----------------|  
|**ClassifiedColumns**|Идентифицирует столбец, описываемый классифицированным столбцом.|  
|**Содержимое**|Тип содержимого столбца.|  
|**Описание**|Описывает столбец. Рекомендуется, чтобы в описании приводилась информация о том, как данные столбца формировались или изменялись для интеллектуального анализа данных.|  
|**DiscretizationBucketCount**|Отображает число контейнеров в дискретизированном столбце.<br /><br /> Включено, только если тип содержимого имеет значение `Discretized`.<br /><br /> Это свойство доступно только для чтения.|  
|**DiscretizationMethod**|Отображает метод, который был использован для дискретизации столбца.<br /><br /> Включено, только если тип содержимого имеет значение `Discretized`.<br /><br /> Это свойство доступно только для чтения.|  
|**Distribution**|Указывает распространение содержимого столбца.|  
|**Идентификатор**|Отображает идентификатор столбца.<br /><br /> Если изменить значение свойства Name столбца, это не повлияет на значение свойства ID.|  
|**IsKey**|Указывает, что столбец является ключевым.|  
|**KeyColumns**|Содержит определение столбца, который является ключевым или входит в состав ключа для атрибута.|  
|**ModelingFlags**|Устанавливает дополнительные параметры, предоставляемые алгоритмом.|  
|`Name`|Имя столбца.|  
|**NameColumn**|Идентифицирует столбец, предоставляющий имя родительского элемента.|  
|**Source**|Отображает источник столбца.<br /><br /> Для реляционных источников данных значением всегда будет **(нет)**.<br /><br /> Для структур, основанных на кубе OLAP, значением является инструкция многомерных выражений, которая определяет срез, используемый в качестве источника для вложенной таблицы.|  
|**SourceMeasureGroup**|Отображает источник группы мер.<br /><br /> Для реляционных источников данных значением всегда будет **(нет)**.<br /><br /> Для структур, основанных на кубе OLAP, значением является инструкция многомерных выражений, которая определяет срез, используемый в качестве источника для вложенной таблицы.|  
|**Тип**|Тип данных для содержимого столбца.|  
  
 Дополнительные сведения о задании или изменении свойств см. в разделе [Задачи и инструкции по структуре интеллектуального анализа данных](mining-structure-tasks-and-how-tos.md).
