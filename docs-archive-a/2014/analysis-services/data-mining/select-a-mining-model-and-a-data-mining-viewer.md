# <a name="select-a-mining-model-and-a-data-mining-viewer"></a>Выбор модели интеллектуального анализа данных и средства просмотра интеллектуального анализа данных
  Модель интеллектуального анализа данных можно изучать с помощью одного из средств просмотра на вкладке **Средство просмотра моделей интеллектуального анализа данных** конструктора моделей интеллектуального анализа данных. Можно легко переключаться между моделями и менять используемое средство просмотра.  
  
-   Раскрывающийся список **Модель интеллектуального анализа данных** на вкладке **Средство просмотра моделей интеллектуального анализа данных** конструктора интеллектуального анализа данных в [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)] содержит все модели интеллектуального анализа, находящиеся в текущей структуре интеллектуального анализа.  
  
-   Для каждого типа моделей предусмотрены специальные средства просмотра. Общие сведения обо всех специальных средствах просмотра см. в разделе [Средства просмотра моделей интеллектуального анализа данных (конструктор моделей интеллектуального анализа данных)](../mining-model-viewers-data-mining-model-designer.md). Пошаговое руководство по использованию специальных средств просмотра для изучения модели см. в разделе [Урок 4. Изучение моделей целевой рассылки (учебник по интеллектуальному анализу данных — начальный уровень)](../../tutorials/lesson-4-exploring-the-targeted-mailing-models-basic-data-mining-tutorial.md).  
  
-   Средство просмотра содержимого общего вида [!INCLUDE[msCoName](../../includes/msconame-md.md)] показывает закономерности, обнаруженные алгоритмом в стандартном представлении узлов дерева. Безусловно, на общем древовидном представлении все содержимое модели приведено с многочисленными подробными сведениями, но задача ее интерпретации является более сложной. Дополнительные сведения см. в разделе [Просмотр модели в средстве просмотра деревьев содержимого общего вида (Майкрософт)](browse-a-model-using-the-microsoft-generic-content-tree-viewer.md).  
  
### <a name="to-select-a-viewer-type"></a>Выбор типа средства просмотра  
  
-   Выберите средство просмотра, предусмотренное для типа созданной модели, в списке **Средство просмотра** в верхней части вкладки **Средство просмотра моделей интеллектуального анализа данных** .  
  
-   Также можно использовать **Средство просмотра содержимого интеллектуального анализа данных (Майкрософт)**  
  
### <a name="to-select-a-mining-model-to-view"></a>Выбор модели интеллектуального анализа данных для просмотра  
  
-   На вкладке **Средство просмотра моделей интеллектуального анализа данных** в конструкторе интеллектуального анализа данных выберите модель, которую необходимо просмотреть из списка **Модель интеллектуального анализа данных** .  
  
 Выбранная модель интеллектуального анализа данных откроется в средстве просмотра, связанном с типом данной модели.
