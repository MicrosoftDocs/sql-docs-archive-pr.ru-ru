  <span data-ttu-id="df6e5-103">В зависимости от типа модели можно использовать детализацию из средств просмотра на вкладке **Средство просмотра модели интеллектуального анализа** в конструкторе интеллектуального анализа данных для изучения вариантов, используемых в модели интеллектуального анализа данных, либо для просмотра дополнительных столбцов в структуре интеллектуального анализа данных.</span><span class="sxs-lookup"><span data-stu-id="df6e5-103">Depending on the model type, you can use drillthrough from the browse viewers on the **Mining Model Viewer** tab of Data Mining Designer to explore the cases used in the mining model or to see additional columns in the mining structure.</span></span> <span data-ttu-id="df6e5-104">Хотя многие типы моделей не поддерживают детализацию, поскольку шаблоны в этой модели нельзя напрямую связывать с конкретными вариантами, приведенные ниже типы моделей поддерживают детализацию.</span><span class="sxs-lookup"><span data-stu-id="df6e5-104">Although many model types do not support drillthrough because the patterns in the model cannot be directly linked to specific cases, the following model types do support drillthrough.</span></span>  
  
 <span data-ttu-id="df6e5-105">Обратите внимание, что детализация в модели должна быть включена и необходимо наличие соответствующих разрешений.</span><span class="sxs-lookup"><span data-stu-id="df6e5-105">Note that drillthrough must have been enabled on the model, and you must have the appropriate permissions.</span></span> <span data-ttu-id="df6e5-106">Функция детализации также должна быть отключена, если модель находится в необработанном состоянии независимо от того, обрабатывалась ли она раньше и есть ли в ней содержимое.</span><span class="sxs-lookup"><span data-stu-id="df6e5-106">The drillthrough option might also be disabled if the model is in an unprocessed state, regardless of whether the model was previously processed and has content.</span></span> <span data-ttu-id="df6e5-107">Для получения данных вариантов модели с помощью детализации, кэш структуры и модели должен быть своевременно обновлен.</span><span class="sxs-lookup"><span data-stu-id="df6e5-107">To retrieve model case data by using drillthrough, the cache of the structure and model must be current.</span></span>  
  
### <a name="use-drillthrough-in-the-microsoft-tree-viewer"></a><span data-ttu-id="df6e5-108">Использование детализация в средстве просмотра деревьев (Майкрософт)</span><span class="sxs-lookup"><span data-stu-id="df6e5-108">Use drillthrough in the Microsoft Tree Viewer</span></span>  
  
1.  <span data-ttu-id="df6e5-109">В конструкторе интеллектуального анализа данных выберите модель дерева принятия решений, затем выберите **Просмотреть модель** , чтобы открыть ее в **средстве просмотра деревьев (Майкрософт)**.</span><span class="sxs-lookup"><span data-stu-id="df6e5-109">In Data Mining Designer, select a decision trees model, and select **Browse Model** to open the model in the **Microsoft Tree Viewer**.</span></span> <span data-ttu-id="df6e5-110">В среде SQL Server Management Studio щелкните правой кнопкой мыши имя модели и выберите **Обзор**.</span><span class="sxs-lookup"><span data-stu-id="df6e5-110">In SQL Server Management Studio, right-click the model, and select **Browse**</span></span>  
  
2.  <span data-ttu-id="df6e5-111">Щелкните правой кнопкой мыши любой узел диаграммы дерева и выберите **Детализация**.</span><span class="sxs-lookup"><span data-stu-id="df6e5-111">Right-click any node in the tree graph, and select **Drill through**.</span></span>
