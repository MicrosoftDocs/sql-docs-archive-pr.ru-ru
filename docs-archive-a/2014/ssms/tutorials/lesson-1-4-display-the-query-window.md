# <a name="display-the-query-window"></a>Отображение окна запроса
  При открытии нескольких окон запросов они отображаются в виде окон документов со вкладками.  
  
## <a name="viewing-multiple-query-windows"></a>Просмотр нескольких окон запросов  
  
#### <a name="to-view-multiple-query-windows"></a>Просмотр нескольких окон запросов  
  
1.  На главной панели инструментов нажмите кнопку **Запрос компонента Database Engine**. В диалоговом окне **Подключиться к компоненту Database Engine** нажмите кнопку **Соединить**.  
  
2.  В обозревателе объектов щелкните правой кнопкой мыши сервер и выберите команду **создать запрос**. Теперь редактор запросов будет использовать сведения о соединении, полученные с зарегистрированного сервера.  
  
     Обратите внимание, что окна отображаются как окна со вкладками справа от обозревателя объектов.
