# <a name="report-manager-f1-help"></a>Справка F1 диспетчера отчетов
  В следующих разделах представлена Справка по диспетчер отчетову на уровне страницы [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] . Каждой странице приложения посвящен отдельный раздел. Доступны могут быть не все страницы. Доступность страниц определяется настройками безопасности. Чтобы открыть эти разделы в диспетчере отчетов, на главной панели инструментов нажмите кнопку **Справка** .
