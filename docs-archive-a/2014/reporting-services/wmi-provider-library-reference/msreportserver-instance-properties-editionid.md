# <a name="editionid-property-wmi-msreportserver_instance"></a>Свойство EditionID (WMI MSReportServer_Instance)
  Возвращает выпуск сервера отчетов. Только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## <a name="property-value"></a>Значение свойства  
 Значение `uint`, которое содержит выпуск сервера отчетов.
