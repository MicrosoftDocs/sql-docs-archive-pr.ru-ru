# <a name="report-sections-dialog-box-report-builder"></a>Диалоговое окно «Разделы отчета» (построитель отчетов)
  При открытии отчета построитель отчетов проверяет, содержит ли отчет только один элемент языка определения отчетов (RDL) **\<Section>** . Открытие отчета с несколькими **\<Section>** элементами не поддерживается.  
  
 Хотя определение отчета с несколькими разделами может быть создано программным путем, изменение отчета в среде подготовки отчетов не поддерживается.  
  
 Чтобы определить текущий формат определения отчета для отчета, сервера отчетов, проекта или используемой среды создания отчетов, см. раздел [Определение версии схемы определения отчета (SSRS)](../reports/find-the-report-definition-schema-version-ssrs.md).  
  
 Чтобы продолжить, нажмите кнопку **ОК** .
